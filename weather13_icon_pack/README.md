# Weather13 Icon Pack

Dieses Paket enthält fertige Wettericons für ein AmigaOS-1.3-Workbench-Programm.

Inhalt:

- `png/`: Vorschau-PNGs und Icon-Sheets für 2bp, 3bp und 4bp
- `c/weather_icons.h`
- `c/weather_icons.c`
- `docs/CODEX_BRIEFING_Weather13.md`

Icons:

- sun
- partly
- cloud
- rain
- storm
- fog
- snow
- unknown

Größen:

- 48x48 für aktuelles Wetter
- 24x24 für 3-Tage-Vorschau

Farbtiefen:

- 2 Bitplanes
- 3 Bitplanes
- 4 Bitplanes

Die C-Daten sind Amiga-planar und enthalten zusätzlich eine 1-Bit-Maske pro Icon.
